# 🌦️ Weather Station Console App

This is a simple C# console application built for **educational purposes** that interacts with a RESTful Weather API. It demonstrates basic usage of `HttpClient`, JSON parsing with `Newtonsoft.Json`, and user input handling in a console environment.

---

## 🚀 Features

- ✅ List all cities with weather information  
- ➕ Add a new city  
- 🗑️ Delete a city by ID  
- ✏️ Update an existing city  
- 🔍 Get city details by ID  

---

## 📦 Technologies Used

- [.NET 6 / .NET 7 Console Application](https://learn.microsoft.com/en-us/dotnet/core/tutorials/)
- `HttpClient` for HTTP requests
- `Newtonsoft.Json` for JSON serialization/deserialization
- Minimal external dependencies, runs completely in the terminal

---

## 🔧 How to Run

1. Clone or download the repository.
2. Make sure you have .NET SDK installed (`dotnet --version`).
3. Open the terminal in the project folder.
4. Run the app with:

```bash
dotnet run
